# StayGi

A full-stack Airbnb clone built with Node.js, Express, MongoDB, and EJS. The application supports user authentication, property listing management, Cloudinary image uploads, and deployment on Render.

## Features

* User Sign Up & Login
* Create, Edit, and Delete Listings
* Image Upload using Cloudinary
* View all property listings
* Responsive UI with Bootstrap

## Tech Stack

Backend: Node.js, Express.js
Frontend: EJS, HTML, CSS, Bootstrap
Database:MongoDB, Mongoose
Cloud: Cloudinary, Render
